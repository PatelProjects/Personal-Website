Introduction

The roots of my passion

My journey as a programmer began when I was introduced to Python in my grade 10 high school computer science class. It truely facinated me. I've always wondered how people gave instruction to computers, applications and other software, and at that moment I realized that through computer science I can not only understand the answer to that at a deeper level but also have the opportunity to be among the few who create tommorrows instructions, for whatever form of technology this era leads us in. Since then, I knew computer science was the road I wanted to take.

Programming Timeline

Major Programming Languages that I use Frequently

I started my programming career in 2015 with python. I learnt the basics in school, such as variable declaration, iteration and basic data structures, but then I went ahead to learn more advance material on my own. Now, Python has become one of the major languages that I use in my side projects and my solutions to popular coding competitions questions and challenges. Some examples of my work include the recreation of the famous game "Connect 4" and a New York Times Web Scraper

In 2017 I was introduced to my second programming language, Java. Many things were similar to Python, and I picked them up quickly, but many things were different as well, so Java came with it's own challenges. Java is the other major language that I use in my side projects. Some examples of what I have created include a file compression/decompression algorithm and an android app that you can find on the play store here.

In the summer of 2018, I began to learn the languages of the web. HTML, CSS and Javascript proved to be very different languages from the others I knew at the time, but I soon found the harmony between them. I've moved on to create projects, such as this website, that include the use of various Javascript libraries such as jQuery, Backbone.js and Underscore.js. Overall, I enjoy the amount of creativity that the languages of the web offer while still demanding the logical aspect of classical programming.
